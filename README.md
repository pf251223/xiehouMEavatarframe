若要在以后的学生节使用，可根据如下步骤更改：
1.将bg.png换成新背景底图，命名保留bg.png；
2.将frame1~6.png换成新头像框，命名照旧保留；
3.logo.png换成新头图，命名照旧保留；
4.编辑index.html:
（1）<footer>换成所需的署名；
（2）<section class="frame-section">中的头像框命名可以修改；
（3）<body>
    <header>
        <img src="logo.png" alt="邂逅ME Logo" class="logo">
        <h1>邂逅ME头像框</h1>
    </header>中的名称可以替换

P.S.请将index.html中的代码复制之后新建仓库进行编辑，以保证本仓库的原始形态；
    所有有关汉字的编辑均可忽略，如果只是想要获得头像框功能（建议更换header中的名称标题）。
